This is a replication file that includes the information necessary to
replicate the results in Stefano M. Iacus, Gary King, and Giuseppe
Porro. "Causal Inference Without Balance Checking: Coarsened Exact
Matching," Political Analysis, forthcoming 2011, copy at
http://bit.ly/cempa.

Complete list of files in the archive:

cemsim-penta.R
cemsim-penta.rda
lalonde-sim.R
measerr.R
measerr2.R
measerr2.rda
progcem.R

Description:

* file measerr.R:
  produces Table 1
  it requires the simulation results in file: measerr2.rda 
  generated by measerr2.R, also available in this folder

* file progcem.R:
  produces Figure 1

* file lalonde-sim.R:
  produces Table 2
  it requires the simulation results in file: cemsim-penta.rda 
  generated by cemsim-penta.R, also available in this folder

* file acompleteone.R:
  produces Figures 2 and 3 and Table 3
